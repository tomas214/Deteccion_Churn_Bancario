# 🏦 Predicción de Churn Bancario

Modelo de clasificación para identificar clientes en riesgo de fuga (**BAJA+2**), optimizando la ganancia económica.

## 🎯 Objetivo de Negocio
* **Acierto (BAJA+2):** +$273,000
* **Error (Falso Positivo):** -$7,000

## 📂 Estructura
* `clase_ternaria.R`: Etiquetado de datos.
* `optimizacion_optuna.py`: Búsqueda de hiperparámetros.
* `entrenamiento_prediccion.py`: Modelo final LightGBM.
